# Contributing

## Scope

This project is a Vite + React + TypeScript codebase for the WPT site.
Contributions should follow the required content order:

`Waste -> Power -> Technology`

## Before You Start

- Review `AGENTS.md` for brand, language, and layout rules.
- Keep text technical, neutral, and short.
- Use the term `molecular disintegration` where the process is described.
- Avoid marketing language and avoid terms such as `burning` or
  `incineration`.

## Development

1. Install dependencies with `npm install`.
2. Run local development with `npm run dev`.
3. Keep changes focused on the requested task.
4. Preserve existing patterns unless the task requires a structural change.

## Quality Checks

Run the relevant checks before opening a pull request:

- `npm run lint`
- `npm run build`
- `npm run test:e2e`
- `npm run test:a11y`

If a command is not relevant to the change, note that in the pull request.

## Pull Requests

- Use a clear title that states the functional change.
- Describe the user-facing or structural impact.
- Include screenshots for visual changes.
- Note any content, translation, or asset updates.
- Reference related issues when applicable.

## Content and UI Rules

- Keep text left-aligned.
- Use Stack Sans for headings and Figtree for body text.
- Use white as the dominant background.
- Use red and blue only as accents.
- Avoid rounded shapes except for the logo.

## Issues

Use the issue templates for bug reports and feature requests.
Provide reproduction steps, expected behavior, and file references.
